#Site web musculation - Projet HTML/CSS

Site web créé pour mettre en pratique les bases de l'intégration web :
HTML, CSS  et design responsive. 

Ce que contient le site :
- Page d'accueil moderne avec explication du site
- Différentes séances (qui me permettent de m'y retrouver de façon claire dans mon programme et d'aider des débutants qui cherchent des exercices de musculation)
- Mise en page responsive
- Typographies modernes
